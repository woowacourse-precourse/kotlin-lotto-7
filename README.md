# 로또

## 💡 기능 요구사항

간단한 로또 발매기를 구현한다.
- 로또 번호의 숫자 범위는 1~45까지이다.
- 1개의 로또를 발행할 때 중복되지 않는 6개의 숫자를 뽑는다.
- 당첨 번호 추첨 시 중복되지 않는 숫자 6개와 보너스 번호 1개를 뽑는다.
- 당첨은 1등부터 5등까지 있다. 당첨 기준과 금액은 아래와 같다.
  - 1등: 6개 번호 일치 / 2,000,000,000원
  - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
  - 3등: 5개 번호 일치 / 1,500,000원
  - 4등: 4개 번호 일치 / 50,000원
  - 5등: 3개 번호 일치 / 5,000원
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 만큼 로또를 발행해야 한다.
- 로또 1장의 가격은 1,000원이다.
- 당첨 번호와 보너스 번호를 입력받는다.
- 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 및 수익률을 출력하고 로또 게임을 종료한다.
- 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시키고, "ERROR"로 시작하는 에러 메시지를 출력 후 그 부분부터 입력을 다시 받는다.
  - Exception이 아닌 IllegalArgumentException, IlleagalStateException 등과 같은 명확한 유형을 처리한다.



## 📋 입출력 요구사항

**입력**
- 로또 구입 금액을 입력 받는다. 구입 금액은 1,000원 단위로 입력 받으며 1,000원으로 나누어 떨어지지 않는 경우 예외 처리한다.
```
14000
```
- 당첨 번호를 입력 받는다. 번호는 쉼표(,)를 기준으로 구분한다.
```
1,2,3,4,5,6
```
- 보너스 번호를 입력 받는다.
```
7
```

**출력**
- 발행한 로또 수량 및 번호를 출력한다. 로또 번호는 오름차순으로 정렬하여 보여준다.
```
8개를 구매했습니다.
[8, 21, 23, 41, 42, 43] 
[3, 5, 11, 16, 32, 38] 
[7, 11, 16, 35, 36, 44] 
[1, 8, 11, 31, 41, 42] 
[13, 14, 16, 38, 42, 45] 
[7, 11, 30, 40, 42, 43] 
[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]
```
- 당첨 내역을 출력한다.
```
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
```
- 수익률은 소수점 둘째 자리에서 반올림한다. (ex. 100.0%, 51.5%, 1,000,000.0%)
```
총 수익률은 62.5%입니다.
```
- 예외 상황 시 에러 문구를 출력해야 한다. 단, 에러 문구는 "[ERROR]"로 시작해야 한다.
```
[ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
```

**실행 결과 예시**
```
구입금액을 입력해 주세요.
8000

8개를 구매했습니다.
[8, 21, 23, 41, 42, 43] 
[3, 5, 11, 16, 32, 38] 
[7, 11, 16, 35, 36, 44] 
[1, 8, 11, 31, 41, 42] 
[13, 14, 16, 38, 42, 45] 
[7, 11, 30, 40, 42, 43] 
[2, 13, 22, 32, 38, 45] 
[1, 3, 5, 14, 22, 45]

당첨 번호를 입력해 주세요.
1,2,3,4,5,6

보너스 번호를 입력해 주세요.
7

당첨 통계
---
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
총 수익률은 62.5%입니다.
```



## ✅ 구현할 기능 목록

### 1. 문자열 입력 받기
- 로또 구입 금액을 입력 받는다.
- 당첨 번호 문자열을 입력 받고 쉼표(,) 기준으로 분리하여 저장한다.
- 보너스 번호를 입력 받는다.

### 2. 결과 출력하기
- 발행한 로또 개수와 로또 정보를 출력 형식에 맞게 출력한다.
- 당첨 기준과 당첨 결과를 출력 형식에 맞게 출력한다.
- 수익률을 출력 형식에 맞게 출력한다.

### 3. 로또 발행하기
- 1에서 45 사이의 숫자 6개를 뽑아 생성한다.

### 4. 수익률 계산하기
- '(당첨 금액 / 구입 금액) * 100' 식을 사용하여 수익률을 구한다.

### 5. 로또 클래스 만들기
- 로또 번호를 저장할 클래스를 생성한다.
- 로또 번호는 6개여야 하고 로또 번호는 중복될 수 없다.
- 로또 번호를 출력할 때에는 오름차순으로 출력되어야 한다.

### 6. 로또 순위 클래스 만들기
- 로또 순위를 저장할 Enum 클래스를 생성한다.
- 일치하는 숫자의 개수, 보너스 번호 일치 여부, 상금, 설명을 저장해야 한다.
- 일치하는 숫자의 개수와 보너스 번호 일치 여부로 당첨 등급을 판별할 수 있어야 한다.

### 7. 당첨 결과 클래스 만들기
- 당첨 결과를 저장할 클래스를 생성한다.
- 로또 순위와 각 순위의 당첨자 수를 저장해야 한다.
- 총 당첨 금액을 계산할 수 있어야 한다.
- 당첨 결과를 출력할 수 있어야 한다.



## 🚀 클래스 설명

### 1. Lotto 클래스
- 기능 : 로또 번호를 저장하는 클래스
- 프로퍼티 : 로또 번호 (List<Int>)
- 초기화 : 로또 번호는 6개여야 하고 로또 번호는 중복될 수 없다.
- 함수
  - getNumbers() : 로또 번호를 리턴한다.
  - toString() : 로또 번호를 오름차순 정렬 후 문자열을 리턴한다.

### 2. LottoRank 클래스
- 기능 : 로또 순위를 저장하는 Enum 클래스
- 프로퍼티 : 일치하는 숫자의 개수 (Int), 보너스 번호 일치 여부 (Boolean), 상금 (Long), 설명 (String)
- 인스턴스 : 1등부터 5등까지 차례로 FIRST, SECOND, THIRD, FOUTH, FIFTH로 요구사항에 맞춰 생성한다.
- 함수
  - from() : 일치하는 숫자의 개수와 보너스 번호 일치 여부로 해당 로또의 당첨 등급을 판별하여 리턴한다.

### 3. LottoResult 클래스
- 기능 : 당첨 결과를 저장하는 클래스
- 해시맵을 사용하여 키로 로또 순위를, 값으로 해당 순위의 당첨자 수를 저장한다.
- 초기화 : 모든 로또 순위의 당첨자 수를 0으로 초기화한다.
- 함수
  - addWinners() : 매개변수로 받은 로또 순위의 당첨자 수를 1 증가시킨다.
  - getTotalPrize() : 총 당첨 금액을 계산하여 리턴한다.
  - showResult() : 출력 형식에 맞게 당첨 결과를 출력한다.



## ✏️ 함수 기능 설명

### 1. inputMoney()
- 기능 : 로또를 구입할 금액을 입력 받는 함수
- 동작 : 금액을 입력 받고 리턴한다. 입력 값이 없는 경우, 입력 값이 숫자가 아닌 경우, 입력한 숫자가 1000으로 나누어 떨어지지 않는 경우 예외 처리한다.
- 리턴 : 구입한 로또의 개수 (Int)

### 2. pickLottoNumbers()
- 기능 : 로또를 발행하는 함수
- 매개변수 : 발행한 로또 리스트 (MutableList<Lotto>)
- 동작 : 1에서 45 사이의 숫자 6개를 무작위로 생성해 발행한 로또 리스트에 추가한다.

### 3. showLottoInfo()
- 기능 : 발행한 로또 정보를 출력하는 함수
- 매개변수 : 발행한 로또 리스트 (MutableList<Lotto>)
- 동작 : 출력 요구사항에 맞춰 구매한 로또 개수와 로또 정보를 출력한다.

### 4. inputWinningNumbers()
- 기능 : 당첨 번호를 입력 받는 함수
- 동작 : 1에서 45 사이의 중복되지 않는 6개의 숫자를 입력 받고 쉼표를 기준으로 분리하여 리턴한다. 입력 값이 없는 경우, 입력 값이 숫자가 아닌 경우, 입력한 숫자의 수가 6개가 아닌 경우, 중복된 숫자를 입력한 경우, 범위 밖의 숫자를 입력한 경우 예외 처리한다.
- 리턴 : 당첨 번호 (List<Int>)

### 5. inputBonusNumber()
- 기능 : 보너스 번호를 입력 받는 함수
- 매개변수 : 당첨 번호 (List<Int>)
- 동작 : 1에서 45 사이의 수 중 하나를 입력 받는다. 입력 값이 없는 경우, 입력 값이 숫자가 아닌 경우, 입력한 숫자가 당첨 번호와 중복되는 경우 예외 처리한다.
- 리턴 : 보너스 번호 (Int)

### 6. checkLottoRank()
- 기능 : 당첨 순위를 확인하는 함수
- 매개변수 : 로또 번호 (List<Int>), 당첨 번호 (List<Int>), 보너스 번호 (Int)
- 동작 : 로또 번호가 당첨 번호와 일치한 숫자의 개수를 구한다. 로또 번호에 보너스 번호가 포함됐는지를 구한다. 두 정보를 통해 해당 로또의 당첨 순위를 구한다.
- 리턴 : 당첨 순위 (LottoRank?)

### 7. checkResult()
- 기능 : 당첨 결과를 확인하는 함수
- 매개변수 : 발행한 로또 리스트 (MutableList<Lotto>), 당첨 번호 (List<Int>), 보너스 번호 (Int)
- 동작 : 당첨 결과를 저장할 LottoResult 객체를 생성한다. 발행한 로또 리스트를 순회하며 모든 로또의 당첨 순위를 구해 LottoResult 객체에 저장한다. 당첨 결과를 출력하고 LottoResult 객체를 리턴한다.
- 리턴 : 당첨 결과 저장 객체 (LottoResult)

### 8. calculateProfit()
- 기능 : 수익률을 계산하는 함수
- 매개변수 : 구입한 로또의 개수 (Int), 당첨 결과 저장 객체 (LottoResult)
- 동작 : '(당첨 금액 / 구입 금액) * 100'의 식을 통해 수익률을 구한다. 출력 요구사항에 맞춰 수익률을 출력한다.

### 9. roundToTwo()
- 기능 : 소수점 둘째 자리에서 반올림하는 함수
- 매개변수 : 반올림할 수 (Double)
- 동작 : 반올림할 수를 소수점 둘째 자리에서 반올림하여 리턴한다.
- 리턴 : 반올림된 수 (Double)
