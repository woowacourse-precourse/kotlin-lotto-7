    - 구입 금액 입력 getPurchasePrice
    - 당첨 번호 입력 getWinningNumbers

- 출력 OutputView
    - 랜덤 생성한 로또 번호 출력 showRandomLottos
    - 로또 일치 개수 출력 showStatus
    - 수익률 출력 showProfitRate
- LottoService
    - ~~당첨 번호 저장~~

    - 최종 수익 구하기 getResultMoney
